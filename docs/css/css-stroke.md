# css描边

[使用text-stroke 和 text-shadow制作CSS描边文字](http://www.htmleaf.com/ziliaoku/qianduanjiaocheng/201705094501.html)

### 方法1
``` css
text-shadow: -1px -1px 0 #000,
               1px -1px 0 #000,
              -1px 1px 0 #000,
              1px 1px 0 #000;
```

### 方法2
``` css
text-stroke: 1px #000;
```
