# 小知识点

**积累踩过的坑，以及零零散散的小知识点等**

> 方面以后快速检索

### 目录结构
```
docs/              docs存放文档，根据模块拆分文档
  mobile/
    ios-scroll.md
  ...
index.html         页面入口
sidebar.md         侧边栏配置
readme.md          入口文档
```
### 如何预览
```
npm i docsify-cli -g
```
在项目根目录
```
npm run dev
```
### 发布
在`master`分支提交代码即可实时更新
